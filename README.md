# JeuMAIN
Le jeu de la MAIN est un jeu où on incarne un jeune étudiant qui arrive à Polytech Sorbonne et intègre la spécialité Mathématiques Appliquées et Informatique. Le but du jeu est de réussir à valider les 3 années du cursus, MAIN3, MAIN4 et MAIN5 tout en surmontant les événements qui surviennent, prédéfinis pour certains, aléatoires pour d'autres.
